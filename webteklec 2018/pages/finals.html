Server - Side Web Scripting

It is a block of codes or program of codes that executes on a web server. They are used in creating dynamic web pages. They are written in a client based language, such as JavaScript, etc. They are embedded in the HTML code in a necessary webpage.


Java Servlets
  • These are Java objects that are intended to play the role of the server components in client - server communications.
  • It handles requests by generating responses to such requests.
  • It is part of the Java EE (Enterpise Edition). The latest platform of the enterprise edition is Java EE8.
  • It is hosted in a 'servlet container' that provides the environment in which the servlet runs, as well as it controls the ‘servlet life cycle’.
